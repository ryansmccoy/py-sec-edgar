# Alignment Plans

This folder contains active alignment plans that need to be addressed.

## Naming Convention

```
YYYY-MM-DD_dimension.md
```

Examples:
- `2025-01-31_documentation.md`
- `2025-01-31_testing.md`
- `2025-01-31_architecture.md`

## Workflow

1. **Create** — AI generates plan from review prompt
2. **Review** — Human reviews and prioritizes
3. **Execute** — Fix issues (AI-assisted or manual)
4. **Complete** — Move to `../completed/` when done
